# React 18 template

## Front

Main part of repo 

### Install

```
cd ./front
npm install
```

### Run build for production

```
cd ./front
npm run build
```

after build you will have `./build` folder with builded files


### Run development server

```
cd ./front
npm run start
```

### Utils


apply prettier for `./src`
```
cd ./front
npm run prettify
```


## Back

Mockserver to provide some backend api


### Install

```
cd ./back
npm install
```


### Run

```
cd ./back
npm run start
```
